# Object-sizes-project
Проект по измерению объектов (книги) с помощью машинного зрения, сравнение использования minAreaRect для измерения и собственного способа расчета с помощью средних значений среди точек на контуре. Смысл в том, что измерение по прямоугольнику дает погрешность от 1 до 2мм, потому что захватывает лишние области помимо контура. 

Интерфейс PyQt5 UI code generator

На картинке: зеленая обводка - контур, синяя обводка - прямоугольник, белые надписи на картинке - измерения по прямоугольнику, измерения снизу справа в окне "Result" - размеры, вычисленные по контуру.
Здесь прямоугольник: 118.5х171.9мм, контур 116.35х170мм, реальные размеры 116.5х160.9мм
